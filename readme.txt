A plugin to add flashcard and quiz functionality to LearnDash topics, compatible with BuddyBoss theme.

== Description == Tora Flashcard allows you to create and manage flashcard sets for LearnDash courses. Features include:





Create flashcard sets and associate them with LearnDash courses.



Display flashcards using the [tora_flashcard] shortcode.



Future support for quizzes (single choice, true/false, match).

== Installation ==





Upload the tora-flashcard folder to the /wp-content/plugins/ directory.



Activate the plugin through the 'Plugins' menu in WordPress.



Go to the Tora Flashcard menu to create flashcard sets.



Use the shortcode [tora_flashcard set_id="X"] to display flashcards.
